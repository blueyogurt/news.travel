レイブパーティだけじゃない!!イビサ島の美しすぎるビーチで、ロマンティックな夕日を眺めよう♪

2016.6.25 remove_red_eye4118

Photo credit: ronsaunders47 via VisualHunt.com / CC BY-SA

「夏といえば、やっぱりパーティー!!」と、ヨーロッパのパーティーピーポーがこぞって向かうスペインのイビサ島。夏だけに限らずイビサ島は一年中毎晩パーティーが行われており、ギネス記録にもなるような巨大なクラブが島内に多数存在しています！

以前そんなイビサ島のクラブやパーティーをご紹介いたしましたが、今回はイビサ島のビーチや、カフェなど、ナイトライフ以外の魅力をたっぷりご紹介いたします！

Photo credit: sky_hlv via VisualHunt / CC BY

まずイビサ島といえば美しいビーチが有名！島中に無数にあるビーチの中でも訪れておくべきビーチとして

▼Cala d’Hort▼
ビーチの目の前に大きな岩島が２つ存在しており、透き通る青い海と岩島のコントラストが大変美しいことで有名なビーチ。イビサ島の観光船も必ず立ち寄る場所で、ビーチレストランはもちろん、マッサージなども受けることができます。

▼Ses Salinas▼
空港から近い、ド定番の南側に位置するビーチ。セレブもよく訪れにくるこちらのビーチは、お洒落なビーチレストランが多く、海はもちろん素敵なランチも楽しめてしまうビーチとして有名です。大きな砂浜で肌を焼くのも良し、レストランで風を浴びながら美味しい食事を楽しむのも格別！

▼Playa d’en Bossa▼
“THEイビサ島”という雰囲気炸裂の有名なビーチ。毎日多くの人で賑わっており、ビーチハウス(いわゆる海の家)や、バーやレストランなども沢山！海の透明度も高く、とってもキレイなビーチなので、イビサ島に来たら一度は訪れたい場所です!!また砂浜では、クラブの宣伝ギャルなども見つけることができるのも、またオモシロイ♪
観光客それぞれビーチの好みも異なるので、イビサ島の無数のビーチの中でお気に入りを見つけてみましょう!!

Photo credit: Michela Simoncini via VisualHunt.com / CC BY

またイビサ島の街並みは、コルドバを彷彿させるような白く大変美しい景観があり、高台にあるダルト・ヴィラ（D’alt Vila）旧市街地は世界遺産にも登録されているんです!!ビーチの後は、イビサ島の可愛い街を散策するのも楽しいですよ！

Photo credit: fotogramaclave via Visual Hunt / CC BY

またイビサ島は美しい夕日を見れる島としても有名で、中でも絶好の夕日スポットとして人気な”Cafe Del Mar”は必ず訪れておきたい場所。美しい夕日にピッタリの音楽と共に眺める夕日は、もう最高としか言いようがありません。多くの観光客を集めるCafe Del Marは、夕日が落ちると一気にパーティームードに早変わり。そのままココで一杯やってから、夕食に出かけるのがイビサ流です！

Photo credit: ST33VO via Visual Hunt / CC BY

ナイトライフだけじゃないイビサ島の魅力はいかがでしたか？パーティーがなくたって十分楽しめるイビサ島、もう寝る間も惜しんで遊ぶしかありません!
今すぐ本当の楽園”イビサ島”に遊びに行きましょう！
