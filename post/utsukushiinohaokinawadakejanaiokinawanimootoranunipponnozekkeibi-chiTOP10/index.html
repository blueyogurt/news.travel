美しいのは沖縄だけじゃない！沖縄にも劣らぬ日本の絶景ビーチTOP10

2016.6.28 スポーツ remove_red_eye1760

【10位】 前浜海岸 / 東京都

出典： www.instagram.com

まず初めに紹介するのは「前浜海岸」です。東京都の神津島にある海岸です。飛行機だと東京から40分、ジェット船だと東京から3時間40分、熱海から1時間30分で着きます。ちょっとした旅行気分を味わいたいのなら、大型客船で東京から11時間、横浜から10時間かけて行くことも可能です。

出典： www.instagram.com

東京にもこんなに透明度の高い海があるなんて、少し驚きですよね。都内からも比較的安く、すぐに行くことができる絶景ビーチです。

出典： www.instagram.com

【9位】 ヒリゾ浜 / 静岡県

出典： www.instagram.com

次に紹介するのは、静岡県の伊豆にある「ヒリゾ浜」。東京からは最短2時間以内（新幹線＋バス）で行くことが可能です。また、車のみで東京から行く場合も約3時間しかかかりません。海沿いをドライブしながら行くのも楽しいかもしれません。

出典： www.instagram.com

海の透明度が高く、景色の良さは折り紙付きです！断崖にさえぎられていて、潮の流れも速いため、中木から渡し船で行くしかないのもまた醍醐味なのです。

出典： www.instagram.com

ヒリゾ浜

【8位】 浄土ヶ浜 / 岩手県

出典： www.instagram.com

次に紹介するのは岩手県宮古市にある「浄土ヶ浜」です。海食・風化によって削られ鋸の歯のような形となった流紋岩が海へ突き出しているのが特徴的です。波が静かな入り江は国内有数の海水浴場になっていて、日本の「快水浴場百選」にも選ばれています！

出典： www.instagram.com

天気がいい日の夜は、夜空一面の星々を見ることができます。普段ゆっくり星を見ることはないのではないでしょうか？訪れた際は、是非この絶景を堪能してください♪

出典： www.instagram.com

スポット情報

浄土ヶ浜

【7位】 水晶浜海水浴場 / 福井県

出典： pixta.jp

次に紹介するのは福井県美浜町の「水晶浜海水浴場」です。「日本の水浴場88選」にも選ばれていて、朝からマリンスポーツを楽しむ人たちで賑わっているのだとか。

出典： www.instagram.com

海の紺碧と白砂、緑が織りなす風景は、まさに水晶そのものです。一度行けばその美しさに圧倒されること間違いないでしょう。きらめく白い砂は、歩くたびにキュッキュッと音がするパウダーサンドです。

出典： www.instagram.com

水晶浜海水浴場

【6位】 白良浜海水浴場 / 和歌山県

出典： pixta.jp

次に紹介するのは和歌山県白浜町にある「白良浜海水浴場 」。約620mにわたる広々とした白砂の海岸は、関西1と言っても過言ではないくらいの人気ビーチです。

出典： pixta.jp

「快水浴場百選」にも選ばれていて、シーズンには、色とりどりのビーチパラソルとヤシの葉が南国ムードを盛り上げます。パッと見ただけだと、日本のビーチだとは思えないくらいの美しさです。浜辺には水着で入浴できる有料露天風呂もあります♪

出典： www.instagram.com

スポット情報

白良浜

【5位】 竹野浜海水浴場 / 兵庫県

出典： pixta.jp

次に紹介するのは兵庫県豊岡市にある「竹野浜海水浴場」。およそ1kmにわたって白い砂浜が続いていて、美しいビーチとしてよく知られています。

出典： www.instagram.com

遠浅で透明度が高い海なので、家族連れにもカップルにも大人気なのです！また、ここ「竹野浜海水浴場は「快水浴場百選」や「日本の渚・百選」にも選ばれています。

出典： www.instagram.com

竹野浜海水浴場

【4位】 角島大橋周辺のビーチ / 山口県

出典： pixta.jp

次に紹介するのは山口県下関市にある「角島大橋周辺のビーチ」。ここ「角島大橋」は、CMなどのロケ地としても使われているため、一度は見たことがあるのではないでしょうか。

出典： www.instagram.com

マリンブルーの海の上に延びる角島大橋は北長門海岸随一の景勝地です。全長1780mで、通行無料の橋としては日本屈指の長さであるんだとか。澄み渡るマリンブルーの色が感動するほど綺麗です。

出典： pixta.jp

スポット情報

角島大橋

【3位】 高浜海水浴場 / 長崎県

出典： pixta.jp

次に紹介するのは長崎県五島市にある「高浜海水浴場」。「日本の渚・百選」や「日本の水浴場88選」、「快水浴場百選」にも認定され、2004年には日本経済新聞で「日本で1番美しい 海水浴場」にも選ばれています。

出典： pixta.jp

この海水浴場は、三井楽半島の西側の付け根に位置しています。三方を山の緑に囲まれ、白銀色の砂浜と透明度抜群の青い海が魅力です。上方の魚籃観音展望所からの眺めも素晴らしいのです。

出典： www.instagram.com

スポット情報

高浜海水浴場

【2位】 永田いなか浜 / 鹿児島県

出典： pixta.jp

次に紹介するのは鹿児島県屋久島の「永田いなか浜」。真っ青な空の青と、海の水色、そして白い砂浜・・・どこを撮っても写真映えしそうですね！

出典： pixta.jp

こちらの海岸では、島内一のウミガメ上陸数を誇っています。およそ1kmにわたる白砂の海岸が美しいと評判です。ウミガメの産卵シーズンは5月から7月にかけてで、見学の際は物音を立てないなど注意する必要があります。

出典： www.instagram.com

永田いなか浜

【1位】 大金久海岸（百合ヶ浜） / 鹿児島県

出典： www.instagram.com

最後に紹介するのは鹿児島県大島郡の与論島にある「大金久海岸」。与論島最大のビーチで白い砂浜が約2kmにわたって続きます。圧倒的な透明度を持っています。まるでハワイを彷彿させる、絶景ビーチなのです！

出典： www.instagram.com

こちらは、大潮の干潮時に大金久海岸の沖に現れる「百合ヶ浜」です。春から夏にかけて沖合い約1.5km地点に大潮の干潮の時間帯にぽっかり姿を現します。潮位が下がるにつれ浜が広がっていくので、様々な表情を見せてくれます。

出典： www.instagram.com
