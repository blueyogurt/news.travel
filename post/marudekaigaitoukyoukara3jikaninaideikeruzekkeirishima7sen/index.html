まるで海外。東京から3時間以内で行ける絶景離島7選

2016.6.27 remove_red_eye929

毎日の喧騒から離れてリラックスしたい♡そんな時にピッタリなのが離島のリゾート。でも海外まで行く時間はないし…というあなた！諦めるのはまだ早いです♪というのも、都内の周辺にある離島なら3時間以内で行けちゃう上に、都内ではまず見られない絶景も見れちゃうんですから♡

ホントに東京!?自然あふれる離島に行ってみよう

via data.whicdn.com
都会の喧騒から離れて、静かなところでゆったりしたい♡毎日忙しい女子の憧れともいえますが、リゾート地は行くには費用も時間もかかってしまう・・。

それなら東京近郊の離島なんていかがですか？

今回は東京からすぐに行けるのにリゾート感満載の絶景離島を紹介します♪あまり知られていない穴場スポットもありますよ♡

夕焼けが感動的な離島

【都内から約45分！】三宅島

via blogs.yahoo.co.jp
せっかく離島に来たのだから、美しい景色が見たいところ。中でも水平線に向かって落ちていく夕焼けは一度見たら忘れられないもの。そんなステキな夕焼けを見るなら三宅島は外せません！

via miyakejima.up.n.seesaa.net
三宅島で最高の夕陽を見るなら、絶対オススメしたいのが三本岳の方向。これだけの美しい光景を見られれば、来てよかったと思えること間違いなしです♡
＜三宅島＞
■住所：
東京都三宅村 三宅島
■アクセス：
羽田空港から飛行機で約45分
詳細はこちら

イルカと泳げる離島

【都内から約80分！】御蔵島

via www.tokyoislands-net.jp
外観を見ると、東京都内とは思えない御蔵島。伊豆諸島の中でも最も水が豊富なことで知られ、源水は甘味があってミネラルがたっぷりと評判。御蔵島では一度は飲んでおきたいですね♪

via stat.ameba.jp
御蔵島の一番の魅力はこれ！なんと野生のミナミバンドウイルカと一緒に泳げちゃうんです♪ダイビングツアーも多く組まれているので、ぜひともチャレンジしてみてください♡
＜御蔵島＞
■住所：
東京都御蔵島村 御蔵島
■アクセス：
羽田空港から飛行機＋八丈島からヘリコプターで約80分
詳細はこちら

青い海がステキな離島

【都内から約105分！】大島

via angie-life.jp
離島に来たなら、やっぱり気になるのが海。下の砂浜が透き通って見えるほどの透明度を誇るのが大島。都内から高速船に乗れば2時間もかからずに到着して、この海で泳げちゃうのは嬉しいですね♪

via www.jalan.net
思いっきり泳いだ後は元町浜の湯という温泉施設がオススメ。海を眺めながらの露天風呂なんて贅沢ですよね♡他にもリス園や三原山などの観光スポットがあるのも大島の魅力です。
＜大島＞
■住所：
東京都大島町 大島
■アクセス：
竹芝桟橋から高速船で約1時間45分
詳細はこちら

サーフィンにおすすめの離島

【都内から約2時間30分！】新島

via blog.goo.ne.jp
大島に負けず劣らずな海のキレイさを誇るのが新島。大島と比べると波がパワフルなので、サーフィンを楽しみたいという方にはピッタリなスポットです♪

via fanblogs.jp
混雑しがちな本土の海水場と比べて、新島のそれは比較的すいているのもポイント。運が良ければコバルトブルーの海をひとり占めできちゃいますよ♪
＜新島＞
■住所：
東京都新島村
■アクセス：
竹芝桟橋から高速船で約2時間30分
詳細はこちら

気分は探検家!?歴史的建造物がスゴイ離島

【都内から30分！】猿島

via find-travel.cdn-dena.com
東京湾に浮かぶ唯一の無人島として知られる猿島。もともとは東京湾を守る要となっていた歴史を誇る島なので、今でも当時の建物がたくさん残っています。

海で泳いだ後にはこうした歴史遺産を観光できちゃうのも猿島の魅力。その雰囲気から「ラピュタのような無人島」とも呼ばれています。
＜猿島＞
■住所：
神奈川県横須賀市猿島1
■アクセス：
京浜急行線 横須賀中央駅から徒歩＆船で30分
詳細はこちら

島だけのグルメが楽しめる離島

【都内から2時間30分！】利島

via blogs.c.yimg.jp

via www.toshimamura.org
せっかくの離島。どうせ行くなら都内で食べられない美味しいものを食べたいところ。そんなグルメな方にピッタリなのが利島。大正時代から盛んになったという漁業で知られていますが、イセエビやサザエがよく獲れることで知られています。新鮮な海の幸を楽しめるなんて最高ですね♪

via bitsubaki.com
また、伊豆諸島に行ったら食べておきたいのが各地名産の野菜として知られる明日葉。独特な風味はヤミツキになる人も♡お蕎麦やお菓子に練り込んだものもあるので、ぜひチャレンジしてみてください♪
＜利島＞
■住所：
東京都利島村 利島
■アクセス：
竹芝桟橋から高速船で約2時間30分
詳細はこちら

一度見たら忘れない♡星空がステキな離島

【都内から約70分！】青ヶ島

via www.arinoki.com
伊豆諸島の中でも秘島とされているのが青ヶ島。八丈島からの乗り継ぎさえうまくいけば1時間程度で到着できますが、時間によってはその日のうちに到着できないというところも秘島っぽさが漂います。

via articleimage.nicoblomaga.jp
携帯も圏外が当たり前な青ヶ島ですが、最大の魅力はこの星空。まるで宝石箱をひっくり返したかのようと形容されるほどの美しい星空はまるで都内とは思えないほど。自然のプラネタリウムを思う存分楽しんでください♡
＜青ヶ島＞
■住所：
東京都青ヶ島村
■アクセス：
羽田空港から飛行機＋八丈島からヘリコプターで約70分
詳細はこちら

大自然に触れて、身も心もリフレッシュ♡

via www.maldives.cx
いかがでしたか？東京から3時間もかからずにこんな大自然のあるところに行けるとは思いもしませんでしたね。東京のもキレイな海や星空がまだまだたくさん残っています。ぜひとも離島へお出かけしてみてはいかがですか♪
