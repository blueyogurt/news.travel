外国人に人気の京都観光地ランキングTOP10

2016.6.24 remove_red_eye3772

トリップアドバイザー株式会社が、2015年の口コミの総数をもとに発表したランキングの中から、外国人に人気のあった京都の観光スポットトップ10をご紹介します。果たして海外ではどういったスポットが人気なのでしょうか？

第10位　嵐山モンキーパークいわたやま（京都市）

第10位は、嵐山モンキーパークいわたやまです。こちらは、100頭以上の野生のニホンザルが生息する場所。実際にサルにエサをあげることもできます。ニホンザルに会えるスポットは日本にもいくつかありますが、外国ではなかなか見られないニホンザルが間近で見られるという点が人気の理由かもしれませんね。

via commons.wikimedia.org
■嵐山モンキーパークいわたやま
・アクセス：阪急嵐山駅より徒歩5分／京福嵐山駅より徒歩5分／JR嵯峨嵐山駅より徒歩15分
・住所：京都府京都市西京区嵐山
・駐車場：なし
・URL： http://monkeypark.jp/

第9位　建仁寺（京都市）

建仁寺の歴史は、1202年に時の将軍源頼家が建立したことがはじまり。以降、火災で焼失することがありましたが、禅の作法を説く禅の道場として栄えていた時代もありました。現在は幾ばくか縮小されたものの、禅のお寺として歴史を紡いでいます。外国人観光客に人気の秘密は、一般の人でも坐禅体験ができるというところ。ほかには、写経体験も行われています。

via www.gettyimages.com
■建仁寺
・アクセス：JR京都駅より車で約10分
・住所：京都市東山区大和大路通四条下る小松町
・駐車場：あり
・URL： http://www.kenninji.jp/
地図を見る

第8位　嵐山（京都市）

第8位は嵐山。嵐山は観光スポットというよりは、西京区を中心としたエリアのことを指します。ここで上げられた嵐山は、嵐山全体の景観のこと。これぞ日本と感じるような竹林や日本橋の風景が、外国人観光客の心をくすぐったのかもしれませんね。ドライブスポットとして、嵐山巡りをしてみるのも良いかもしれません。

via www.gettyimages.com
■嵐山
・住所：京都府京都市右京区
・URL： http://www.arashiyamahoshokai.com/
地図を見る

第7位　京都駅ビル（京都市）

第7位には、なんと京都駅ビルがランクイン。ショッピングからグルメ、そして劇場にお土産と何でもそろうのが人気の秘密でしょうか。中でも、京都のお土産が一気に揃うという点が、外国人観光客からすると評価される点なのかもしれませんね。

via commons.wikimedia.org
■京都駅ビル
・アクセス：八条通り京都駅内
・住所：京都府京都市下京区東塩小路釜殿町
・駐車場：あり
・URL： http://www.kyoto-station-building.co.jp/index.htm
地図を見る

第6位　三十三間堂（京都市）

三十三間堂は、国宝にもなっている京都の観光スポットです。外国人観光客だけでなく、日本人にも人気の場所。さらに三十三間堂には、国宝にもなっている千手観音坐像が安置されています。なかなか見られないありがたい菩薩が見られるのが人気の秘密かもしれませんね。

via www.gettyimages.com
■三十三間堂
・アクセス：京阪七条駅より徒歩約7分
・住所：京都市東山区三十三間堂廻り町657
・駐車場：あり/50台
・URL： http://sanjusangendo.jp/
地図を見る

第5位　清水寺（京都市）

山の中腹に建てられた清水寺は、国内でも良く知られた観光スポット。実は海外でも良く知られたスポットです。理由は、古都京都の文化財として世界文化遺産に登録されているため。世界が認めるだけあって、境内には国宝や国の文化財などとして名を連ねる複数の伽藍が立ち並んでいます。

via www.gettyimages.com
■清水寺
・アクセス：京阪電鉄清水五条駅より徒歩25分
・住所：京都府京都市東山区清水1丁目294
・駐車場：なし
・URL： http://www.kiyomizudera.or.jp/
地図を見る

第4位　禅林寺 永観堂（京都市）

絹本着色山越阿弥陀図や金銅蓮華紋磬など58点の国宝級の文化財が残るお寺。こちらも外国人観光客に人気の高いスポットです。国宝が多数所蔵されているというのが人気の秘密でしょうか。秋になると美しい紅葉が見られるので、ドライブスポットとしてもおすすめの場所ですよ。

via www.flickr.com
■禅林寺 永観堂
・アクセス：JR京都駅から市バス5系統で「南禅寺永観堂道」下車、徒歩3分
・住所：京都府京都市左京区永観堂町48
・URL： http://www.eikando.or.jp/
地図を見る

第3位　鹿苑寺 （金閣寺）（京都市）

室町時代の将軍、足利義満によって創建された金閣寺は、歴史の教科書にも載るほどあまりにも有名な建物です。書院造という質素だけれども厳格な造りは、平安時代の貴族文化とは全く雰囲気の異なるものです。金閣寺の美しさと、周囲の日本庭園の美しさが外国人にも人気の理由でしょう。

via www.gettyimages.com
■鹿苑寺 （金閣寺）
・アクセス：市バス「金閣寺道」よりすぐ
・住所：京都市北区金閣寺町1
・URL： http://www.shokoku-ji.jp/k_about.html
地図を見る

第2位　サムライ剣舞シアター（京都市）

あまりなじみがないかもしれませんが、外国人観光客に熱いスポットがサムライ剣舞シアター。正当な剣舞の指導者正賀流剣舞師範を筆頭に剣舞が体験できるというものです。着物を着て、刀を振り回す様は、まさにサムライそのもの。日本＝サムライの国と考える観光客もまだまだ多いため、観光客の心を良くつかんだスポットだと言えます。
■サムライ剣舞シアター
・住所：京都市東山区三町目35-7 ネオアートビルBF
・URL： http://www.samurai-kembu.co.jp/
地図を見る

第1位　伏見稲荷大社（京都市）

そして、堂々の第1位に輝いたのが伏見稲荷大社。お稲荷さんを祭った神社で、幾重にも連なった鳥居の風景が圧巻です。外国人観光客には、この鳥居の風景が幻想的で日本的なスポットとして映ったのかもしれません。

via www.gettyimages.com

・住所：京都市伏見区深草薮之内町68番地
・駐車場：あり
・URL： http://inari.jp/
