横川サービスエリアのおすすめグルメ5選

2016.6.25 remove_red_eye3417

上信越自動車道の人気サービスエリアと言えばやはり横川サービスエリアです。日本一人気の駅弁とも言われる「峠の釜めし」他、魅力的なグルメが盛りだくさん。立ち寄ったらぜひ食べておきたいおすすめグルメを、厳選してご紹介します。

横川サービスエリアのおすすめグルメが知りたい！

上信越自動車道にある横川サービスエリア。横川といえば「釜めし」を思い浮かべる方も多いのではないでしょうか。その釜めしで有名な信越本線の横川駅が2009年のリニューアルで再現されました。釜めしで有名なだけあり、エリア内にはちょっと珍しい「空き釜入れ」なる大きな回収箱も用意されています。ですが、横川サービスエリアの名物は釜めしだけではありません。群馬でしか食べられない名物グルメが他にもたくさんあります。横川サービスエリアに立ち寄ったらぜひ食べておきたいおすすめグルメをご紹介します。

1.峠の釜めし本舗おぎのやの「峠の釜めし」

横川サービスエリアの代名詞ともいえるのが「おぎのや」の「峠の釜めし」。この土釜を見たことがあるという方も多いのではないでしょうか。益子焼の土釜に入ったこの釜めしは、昭和33年から続いている人気商品です。日本一人気の駅弁とも言われ、この釜めしを目当てに訪れる方も少なくありません。空き容器が必要ない場合は回収ボックスに入れることができますが、ご自宅に持ち帰って活用している方も多いようです。1人分の炊き込みご飯やおかゆ、鍋焼きうどんなどが作れますよ。横川サービスエリアにお立ち寄りの際はぜひ！

via ja.wikipedia.org

2.たかべんの「だるま弁当」

高崎弁当、通称「たかべん」が販売する高崎名物だるま弁当。その名の通り、だるまの容器に入ったお弁当で峠の釜めしと人気を二分しています。販売開始はなんと昭和35年。当初は瀬戸物の容器でしたが、現在ではプラスチックのだるま型容器となっています。食べ終わった後は貯金箱として再利用も可能です。肝心の中身ですが、しょうゆ味のご飯の上に山菜やしいたけ、鶏肉などがたっぷり入ってとてもヘルシー。世代を問わず親しまれているお弁当です。

via ja.wikipedia.org

3.Backerei Gluck（ベッカライグリュック）

横川サービスエリア下り線のおいしいパン屋さん「Backerei Gluck（ベッカライグリュック）」。店内に入るとまず目に入るのがインパクトのあるだるまのパン。人気商品のひとつ、リンゴフィリング入りのかわいい「だるまくん」です。食べるのがもったいないようなかわいさですが、味もお墨付き。そのほかにも富岡製糸工場にちなんだ蚕の形の白いパン「おかいこbread」や、かわいい虎の「トラカレーパン」など見た目にも楽しいパンが並びます。「おかいこbread」の味は、チョコレート・抹茶・ピーナッツクリームの3種類。甘いパンばかりではなく、コロッケサンドなど食事系のパンもそろっていますので、朝食にもぴったりです。朝は7時から開店しています。
