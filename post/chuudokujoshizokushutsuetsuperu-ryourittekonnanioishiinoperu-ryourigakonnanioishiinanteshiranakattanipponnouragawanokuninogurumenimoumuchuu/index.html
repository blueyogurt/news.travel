中毒女子続出！えっ!!ペルー料理って、こんなに美味しいの?! ペルー料理がこんなに美味しいなんて知らなかった！日本の裏側の国のグルメにもう夢中♡

2016.6.26 remove_red_eye971

Photo via VisualHunt.com

日本は美食の国として有名ですが、今や国内で様々な国のグルメを楽しめるようになりました。イタリアンや、中華などは誰でも食べたことのあるものですが、例えば遠く離れた南米の料理と聞いて、どんなものを思い浮かべますか？南米の中でも、最近はブラジル料理もポピュラーになってきましたが、私の一番のオススメは断然、ペルー料理!!
正直なところ、初めて食べるまでは敬遠していたのですが、一度食べれば、もう中毒レベル。美味しすぎて、今まで敬遠していたことを後悔してしました。
未体験のあなたに、私のオススメペルー料理をご紹介させて頂きます！

まずはド定番”セビチェ(ceviche)”

Photo credit: Blok 70 via Visualhunt / CC BY

新鮮な白身魚、タコ、エビ、イカなどレモンと酢であえ、最後に玉ねぎをのせた絶品の海鮮マリネです。お店によって種類なども様々ですが、さっぱりとした日本人好みの味つけでファン急増中！ちなみに私もペルー料理のレストランに行けば必ずセビチェをオーダーしています!!

醤油ベースの炒めもの”ロモ・サルタード(Lomo Saltado)”

Photo credit: Nagyman via VisualHunt / CC BY-SA

こちらは中華料理風の一品で、牛肉の細切り、玉ねぎ、トマト、フライドポテトを炒めたペルーのソウルフード！醤油ベースで、我々日本人の口にもとても合うんです！旅先で日本食シックに陥るのはよくある話ですが、ペルーにおいては繊細な日本人も現地のローカルグルメを楽しみすぎてしまいます。

“チュペ・デ・カマロネス(chupe de camarones)”

Photo credit: _e.t via VisualHunt / CC BY-SA

こちらは川エビを煮込んだスープ！エビの濃厚なダシに牛乳を混ぜ、ハーブで味つけしたスープは絶品です。特産品であるジャガイモが具に入ったチャウダー風スープ。海鮮に恵まれているペルーでは、海の幸をふんだんに使った料理が多く、そういった点も日本人の胃袋を掴んで放さないのかもしれませんね。

“インカ・コーラ(INCA KOLA)”

Photo credit: Matito via Visual Hunt / CC BY-SA

ペルーといえば、コレ！アルコールではなく、エナジードリンク風のサイダーなんです！その黄金の輝く色合いは唯一無二！子供からも大人からも愛されるペルーを代表するドリンク。ペルー料理のレストランでは、アルコール以外はみんなコレを頼んでたりします。一度飲めばハマること間違いなし！

ペルーは様々な民族の食文化が混交してきた場所であり、他国では見ることのできない独特の料理が多く存在しています。中国系・日系のルーツも含むペルー料理は、日本人にとっても親しみ易く、またどこか懐かしさまで感じる絶品料理。
未体験のアナタ！本場ペルーまで行くのが厳しければ、是非近場のペルーレストランを、まずは予約してみましょう！
