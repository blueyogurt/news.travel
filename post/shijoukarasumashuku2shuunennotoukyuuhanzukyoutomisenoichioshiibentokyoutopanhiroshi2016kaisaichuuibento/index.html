四条烏丸＊祝2周年の「東急ハンズ 京都店」のイチオシイベント「京都パン博 2016」開催中！【イベント】

2016.6.24 remove_red_eye1373

6/19で開業2周年を迎えた東急ハンズ京都店。2周年を記念してさまざまなイベントが開催中ですが、その中でも注目度抜群のイベントが只今行われていますよ〜

パン好きのためのイベント！「京都パン博 2016」

via kyoto.tokyu-hands.co.jp

via kyoto.tokyu-hands.co.jp

via kyoto.tokyu-hands.co.jp
京都ハンズ開業2周年記念に合わせて行われるこのイベントは、京都の有名パン処16店が一同に集結。
老舗店から新鋭店まで、バラエティに富んでいてかなり魅力的。

今日、6/24から始まっていますが、開店前から行列ができていたそう。
そして人気のパンは午前中で完売する盛況ぶりで、さすがはパン消費量日本一の京都。皆さん目ざといです(笑)。
明日、明後日も混雑が予想されますので、お目当てのパンがある方はお早めに来店、GETしてくださいね！

個人的には「アペリラ」と、「雨の日も風の日も」が気になります！

※店舗により出店日が異なりますので、詳しくはご確認ください※

via kyoto.tokyu-hands.co.jp
また『地元パン手帖』の著者・甲斐みのりさんがセレクトした「地元パン」の限定販売と、著書に掲載されているパンにまつわる写真も展示もあるそうです。
滋賀のあのパンもあるみたい？こちらもめっちゃ貴重です！

同時開催「ENJOY COFFEE TIME」

via http://kyoto.tokyu-hands.co.jp/item/2nd-anniversary2.html

via kyoto.tokyu-hands.co.jp
パンと一緒においしいコーヒーを、という事で、3階の「コーヒーアワー」を中心に「ENJOY COFFEE TIME」と銘打ったコーヒーイベントも同時開催中。
普段からたくさんの種類の豆が販売されていてスタッフさんも知識豊富、そしてコーヒースタンドとしてコーヒーを飲むことができますが、通常は取り扱いのない豆を売られており貴重な期間になっています。

個人的にオススメなのは「大山崎COFFEE ROASTERS」。焙煎して3日以内の新鮮な豆、しかもシングルオリジンにこだわって販売されていて、とてもおいしいです。
6/26日曜日には、ワークショップ（受付終了）や試飲イベントもありますので、この機会に是非大山崎の豆に触れてみてくださいね。

先日立誠小学校で行われたコーヒーイベントと同じネーミングですが、実は東急ハンズ京都店が主催していたんです。なので出店ラインナップなど、関連イベント的な感じになっています。

明日6/25土曜日はUnirのコーヒーが試飲できるそうですし、「PAINLOT」プロデュース、パンとコーヒーのワークショップが開催されるとの事。
”コーヒーに合う美味しいパンの試食とスペシャルティコーヒー「Unir」による
アイスコーヒーの試飲、京都パン通がこっそり教える“パンの美味しいカフェ”をご案内します。”
という内容だそうで、パン好き、カフェ好きにはたまらない内容な予感。
残席わずかですが参加可能な会があるようですので、気になる方はチェックしてみてください。
http://happy-event.tokyu-hands.co.jp/event/index.php?event_id=3632&store_id=27&sec=15534087ldjttcl6ogc01bkf008pctv1
その他エントランスでは、今年10周年を迎える「中村風鈴店」の売り歩き風鈴販売や、「タカトモハンコ」のはんこが押せるというスタンプラリーも開催されていますので、あれやこれや2周年をお祝いしつつ楽しんでみてはいかがでしょうか。
