ハリウッドスターも御用達！ヨーロッパの人々が憧れる極上リゾート『マルタ島』って、どんな所？

2016.6.27 remove_red_eye2371

Photo credit: flavouz via VisualHunt / CC BY

イタリアから、南へ100キロ程の地中海に浮かぶ小さな島『マルタ島』。
地中海クルーズでは必ず立ち寄る寄港地として知られていますが、日本人にはまだまだ馴染みのない場所ですよね。ヨーロッパの人々も憧れるリゾート地として有名で、実はハリウッドスターもお忍びで羽を伸ばしにきています。

Photo credit: trishhartmann via Visualhunt.com / CC BY

マルタ島が人気である理由の一つに治安の良さが挙げられ、バカンスそして留学先としても選ばれるマルタ島は、常識範囲内での深夜の外出なども全く問題なく、女の子だけでも安心して過ごせるリゾート地！また物価も安く、お財布を気にせず思う存分休暇を楽しめる最高の場所なのです。

Photo credit: Manlio Ranieri via Visualhunt / CC BY-SA

ちなみに島内のセント・ジュリアンは、マルタ島を代表する大人気の繁華街！やっぱりバカンスには、ナイトライフも楽しみたいというアナタに、お洒落なバーやクラブが沢山ある、こちらの街は必須！治安の良いマルタ島では、朝まで多くの人で賑わっています。ここセント・ジュリアンは高ランクのホテルも多く、また美味しいレストランも沢山あります。ちなみにハリウッドスターも訪れるレストラン”La Dolce Vita”は眺めも、食事も素晴らしいので要チェック！

Photo credit: David Locke via Visual Hunt / CC BY

肝心のビーチですが、同じくマルタ共和国内コミノ島”ブルー・ラグーン”は絶対に行くべき絶景スポット！世界的なダイビングスポットとしても有名で、その美しすぎる海は見てるだけでも癒されてしまいます。信じられないようなエメラルドブルー！ちなみに、マルタ島北部のメーリッハ湾やゴールデン湾も大人気のビーチなので、時間の許す限りビーチでリラックスしちゃいましょう！

Photo credit: Heini Samuelsen via VisualHunt.com / CC BY-SA

気になるグルメですが、マルタ島は料理も美味しく安い！(量も多いんです。)
イタリアの影響を色濃く受けつつも、オリジナリティ溢れる料理を楽しむことができるマルタでは、例えば伝統料理”ブラジオリ”や、タコも特産品なので是非とも食べて頂きたいところ。魚介を使ったパスタに、美味しいイタリアンワインで乾杯なんてのも、オススメです♪

Photo credit: Arkangel via Visual Hunt / CC BY-SA

ヨーロッパの人々も憧れる理由が、マルタ島には盛り沢山。まだまだ日本人にはマイナーな観光地ですが、逆に行くなら今かもしれません！さぁ、マルタ島への準備を始めてみませんか？
