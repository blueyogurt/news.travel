ゲームの世界が実現！「リアルマリオカート」って知ってる？

2016.6.28 芸能 remove_red_eye3150

マリオシリーズで人気のマリオカート。実は、ゲームの中だけでなく、実際に自分でリアルなマリオカートを体験できるのをご存知でしょうか。今回は、そんな夢のようなマリオカートの世界が現実化したリアルマリオカートをご紹介します。

マリオカートの世界が現実化！？

リアルマリオカートとは、マリオカートが現実化したもののこと。東京都内を中心に、日本各地、そして海外などでもリアルマリオカートがたびたび目撃されているようです。ゲームの世界が現実化しているなんて、なんともコミカルで幻想的ですね。

via www.flickr.com

ちょっと待った！そもそもマリオカートとは？

リアルマリオカートがゲームの世界を飛び出したような感じなのかは、ある程度イメージできたかと思います。しかし、そもそもマリオカート自体が何なのか分からないという方も含めて、一度おさらいしてみましょう。マリオカートとは、スーパーマリオシリーズで知られる、マリオやルイージ、ヨッシ―、ピーチ姫などのキャラクターで構成されたアクションレースのゲームです。プレイヤーは好きなキャラクターを選択して、レースに臨みます。他のレーシング系のゲームと異なるのが、バナナや爆弾などのアイテムを駆使して相手を邪魔したり、マリオシリーズならではのファンタジー感あふれるステージが設けられているという点です。こういった世界観が、子どものみならず、大人や女性などさまざまな層で人気を集めてきました。

via www.flickr.com

リアルマリオカートのあの車の正体とは...

ゲームの世界観を復習したところで、もう一度リアルマリオカートに戻ってみましょう。やはり気になるのは、なんといってもあの車体。リアルマリオカートは公道で走っている姿をたびたび目撃されていますが、違反車ではありません。実は、公道でも走ることができる専用のカートで、ウィンカーなどの基本装備はもちろん、ヘルメット、そしてシートベルト着用の必要がないものとなっています。道路交通法では、ミニカーと分類されるからです。
ミニカーに分類される車体ということは、当然特別な免許なども必要ありません。普通自動車免許を所持していれば、誰でもリアルマリオカートを運転することができるのです。もちろん、オートマ限定でも大丈夫。自動車免許さえ持っていれば運転できるとなると、なんだか夢が広がりますね。

via www.flickr.com

自分で所持していなくても大丈夫！マリオカートはレンタルできる

ただ免許の面でクリアできたとしても、やはり車体をどうやって手に入れるかということ。例えば、リアルマリオカートとして使われるX-kartの値段は税抜きで395,000円となっています。趣味レベルで言えば少々お値段が張るというのもありますし、そもそも都内であれば、カートの置き場所の問題も発生してしまいますよね。しかし、そんな悩みも解決してくれるのが、レンタルサービス。レンタルで使用すれば、気が向いたときに気軽にリアルマリオカートを楽しむことができます。

via www.gettyimages.com

レンタルショップ「MARICAR」

都内でも大きなレンタルショップが、「MARICAR」。1人乗りのカートX-kartのレンタルを行っているショップです。魅力は、コスプレの貸出があるということ。マリオやルイージ、ヨッシ―などコスプレでキャラクターになりきって楽しむことができます。さらに、通常55kmが限度のカートですが、高速チューニング済みなのでさらに速い速度で走行することが可能となっています。
