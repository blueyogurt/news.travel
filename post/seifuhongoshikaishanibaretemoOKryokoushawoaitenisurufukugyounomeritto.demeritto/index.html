政府本腰！会社にばれてもOK！？【旅行者】を相手にする副業のメリット・デメリット

2016.6.25 remove_red_eye4428

「今月お金を使い過ぎて大ピンチ！」「給料日までまだ先が長いのにどうしよう……」

そんなときにふと頭をよぎるのが節約や副業ですが、節約したからといってすぐにお金が沸いてくるわけでもないし、株や FX をやろうにも元手が必要だし、知識やノウハウを知っているわけでもない……。

「誰か楽して稼げるいい方法教えてーー」と天を仰ぎたくなりますよね。でも、じつは最近、特別なスキルも元手もいらず、楽して稼げる方法があるのをご存じですか？

それが「 Airbnb 」（エアビーアンドビー）です。

https://www.airbnb.jp/

新たな副業の手段「 Airbnb 」（エアビーアンドビー）とは

これは、宿泊地を探している旅行者（ゲスト）と宿泊地の提供者（ホスト）をつないでくれる Web 上のマッチングサービス。自宅や自宅の一室、空き物件などを貸すことで、ゲストから宿泊費をもらえる仕組みで、０８年に米国で始まるやいなや、利用者が急速に増え、いまや１９０カ国以上、のべ６０００万人が利用しているほどの人気ぶりなのです。日本でも多くの雑誌や新聞、テレビでも様々なかたちで紹介されて人気になっています。

政府も本腰

とくに、日本は２０２０年の東京五輪開催に向けて、政府が本腰を入れて、インバウンド需要の囲い込みに力を注いでいる背景もあり、「 Airbnb 」で稼ごうと思っている人にとっては追い風が吹いているといえます。実際、学生や主婦の中には、副業で数十万円を稼ぐ強者も出てきているなど、ここ最近注目度が高まっている副業の一つといえるのです。


※2016年3月末には、当初目標としていた訪日外国人観光客数の目標人数を倍増させ、平成３２（２０２０）年に４千万人、４２（２０３０）年に６千万人としています。

具体的な利用方法

じつは、これもいたってシンプルです。

まず Airbnb サイトから「ホスト」として物件を登録。ホストの顔写真をアップロードして、プロフィール＆紹介文を掲載。料金や部屋など宿泊場所の詳細をゲストがわかるように紹介するだけです。あとはゲストからの予約を待って、承認するかどうか。

田舎も人気！？

「 Airbnb 」を利用して日本を訪れる人のほとんどは外国人旅行者。それも日本の文化や歴史に興味があるという親日家が多いといいます。それゆえ、たとえ、都心から離れた場所だろうと、田舎だろうと、ゲスト側が魅力を感じてくれれば、利用される可能性が出てくるといえるのです。


最近では、農業体験ができる古民家も人気とのことです。

危険性

ただし、見ず知らずの外国人を招くのですから、利用する際には注意も必要です。

ゲストが部屋に損害を与えてしまった場合は、 Airbnb が損害補償してくれるので、お金の心配はいりませんが、独身女性が「ホスト」として自宅などを提供する際は犯罪に巻き込まれる危険性も十分あるので、親や友人に協力してもらうなど、利用の際は細心の注意を払うようにしましょう。
