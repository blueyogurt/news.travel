箱根ドライブで行きたい観光スポット10選

2016.6.28 remove_red_eye1357

首都圏から1時間少しでアクセスでき、富士箱根伊豆国立公園の豊かな自然や数多くの観光スポット、そして温泉と魅力あふれる箱根。風景も美しくドライブにはおすすめのルートです。今回は、箱根ドライブの際に訪れたい観光スポットをご紹介します。

大涌谷

via www.thinkstockphotos.com
標高1044mにある大涌谷は、約3000年前の大噴火の際に山崩れが起こった場所に、さらに火山噴火による堆積物が積もってできた地形です。地面の割れ目のあちらこちらから噴煙が上がっている景色を見ることができます。大涌谷の温泉池で茹でた「黒たまご」が名物です。
■大涌谷
・アクセス：東名高速道路「厚木」ICから約30分、「御殿場」ICから約40分
・住所：足柄下郡箱根町仙石原1251-1
・駐車場：あり
・URL： http://www.kanagawa-park.or.jp/owakudani/index.html
地図を見る

箱根関所

via photozou.jp
江戸の防備のために重要な位置付けであった箱根関。当時の技術や道具を使って、箱根関跡に江戸時代後期の京口御門・大番所・上番休息所・厩（うまや）の姿が約140年ぶりに完全復元されました。併設する箱根関所資料館では、箱根の関所に関する資料や完全復元の様子などが展示されています。
■箱根関所
・アクセス：小田原厚木道路「小田原西」ICから約40分、東名高速道路「御殿場」ICから約50分
・住所：神奈川県足柄下郡箱根町箱根1
・駐車場：あり
・URL： http://www.hakonesekisyo.jp/index.html
地図を見る

芦ノ湖

via www.thinkstockphotos.com
芦ノ湖は、約3100年前に水蒸気爆発によって神山が崩れ、土砂が川をせき止めたことによってできました。湖底からの湧水が水源のため、冬でも凍結しない湖です。背後に富士山を望む、箱根の代表的な景勝地として親しまれています。湖畔には、九頭竜神社・箱根関所・箱根園・箱根公園など多くの観光スポットが点在しています。
■芦ノ湖
・アクセス：東名高速道路「御殿場」ICから約50分、「厚木」ICから約60分
・住所：神奈川県足柄下郡箱根町元箱根 芦ノ湖
・URL： http://www.hakone-ashinoko.net/index.html
地図を見る

箱根駒ヶ岳ロープウェー

via ja.wikipedia.org
箱根駒ヶ岳ロープウェーは、芦ノ湖畔に位置する箱根園から標高1327mの箱根駒ケ岳の山頂までを約7分で結ぶ全長1783mのロープウェイです。芦ノ湖・富士山・箱根外輪山などの絶景を楽しみながら山頂へ登れます。山頂には箱根神社の奥宮・箱根元宮がある他、相模湾・伊豆七島・房総半島などを、望む大パノラマを堪能できます。
■箱根駒ヶ岳ロープウェー
・アクセス：東名高速道路「厚木」ICから約65分、「御殿場」ICから約45分
・住所：神奈川県足柄下郡箱根町元箱根139
・駐車場：あり（箱根園の駐車場を利用）
・URL： http://www.princehotels.co.jp/amuse/hakone-en/watch/ropeway_enjoy.html
地図を見る

箱根小涌園ユネッサン

via photozou.jp
箱根小涌園ユネッサンは、箱根小涌園の跡地に建設された全天候型の温泉を利用したテーマパーク。水着入浴エリアの「ユネッサン」・裸入浴エリアの「森の湯」とがあります。広々とした泳げるスパの他、ワイン風呂・緑茶風呂・コーヒー風呂・坂風呂などさまざまな種類の浴槽があります。
■スポット名を入れてください
・アクセス：東名高速道路「厚木」ICから約60分、「御殿場」ICから約40分
・住所：神奈川県足柄下郡箱根町二ノ平1297
・駐車場：あり（1000台）
・URL： http://www.yunessun.com/
地図を見る

箱根神社・九頭竜神社

via www.gettyimages.com
757年に創建された神社で、もともとは「関東総鎮守箱根権現」として崇拝されていました。箱根駒ヶ岳の山頂には奥宮である「箱根元宮と宮」が鎮座しています。九頭竜神社は箱根神社の末社の1つで、箱根神社境内にある新宮には霊水「龍神水」が湧いています。箱根神社・元宮・九頭竜神社を合わせて参拝すると、健康・恋愛・出世・金運などバランスよく運をアップできると言われています。
■箱根神社・九頭竜神社
・アクセス：東名高速道路「沼津」IC・「御殿場」ICからそれぞれ約60分
・住所：神奈川県足柄下郡箱根町元箱根80-1
・駐車場：あり
・URL： http://hakonejinja.or.jp/

星の王子さまミュージアム

via ja.wikipedia.org
星の王子さまの作者・サン=テグジュペリの生誕100周年の記念事業の一環で1999年オープンした「星の王子さまミュージアム」。敷地内には、1990年代のフランスを再現した街並みや季節の花々が咲くガーデンなどが広がっています。展示ホールにはサン=テグジュペリが過ごした部屋や手紙・写真などが展示されています。
■星の王子さまミュージアム
・アクセス：東名高速道路「御殿場」ICから約20分
・住所：神奈川県足柄下郡箱根町仙石原909
