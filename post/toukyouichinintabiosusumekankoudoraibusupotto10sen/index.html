東京一人旅 おすすめ観光ドライブスポット10選

2016.6.26 remove_red_eye2110

東京は都会的な一面もあれば、自然がたっぷりの場所もたくさんあります。何よりも歴史が深いのでマイペースに江戸情緒の雰囲気を楽しんだり散策してみたりすることも楽しいですよね。そんな一人旅でも楽しめる東京のドライブスポットを紹介します。

東京スカイツリー（墨田区）

via www.thinkstockphotos.com
東京スカイツリーは高さ634mの電波塔であり、東京の新しい観光スポットとしても大人気。展望台からは360度の東京の大パノラマが楽しめます。併設する東京ソラマチでのショッピングモールには外国人観光客向けの日本土産のお店もたくさんありますし、「すみだ水族館」や「コニカミノルタプラネタリウム」などの施設も楽しめます。
■東京スカイツリー
・アクセス：首都高速6号向島線向島から1分
・住所：東京都墨田区押上1丁目1-2
・駐車場：あり（有料）
・URL： http://www.tokyo-skytree.jp/

若洲海浜公園（江東区）

via www.gettyimages.com
2012年に開通したばかりの東京ゲートブリッジが良く見える公園。とくに夜景はとても素敵なので、カメラを持って訪れたいスポット。昼間も釣りやアウトドアキャンプ、サイクリングなどを楽しむことができます。
■若洲海浜公園
・アクセス：首都高速湾岸線新木場ランプより東京へリポート方向へ7分
・住所：東京都江東区若洲三丁目2番1号
・駐車場：あり
・URL： http://www.tptc.co.jp/park/03_07
地図を見る

大江戸温泉物語（江東区）

via www.gettyimages.com
レインボーブリッジでお台場ドライブ。ダイバーシティお台場やパレットタウンなどたくさん遊んだあとは大江戸温泉物語でゆったりリラックスして過ごしたいですね。お風呂のあとは食事処でゆっくりしたり、エステやあかすり、岩塩風呂などで疲れを癒しましょう。
■大江戸温泉物語
・アクセス：首都高速11号台場線台場出口から約3分
・住所：東京都江東区青海2丁目6番3号
・駐車場：普通車216台（4時間まで1000円）
・URL： http://www.ooedoonsen.jp/daiba/index.html
地図を見る

江戸東京博物館（墨田区）

via www.gettyimages.com
江戸と東京について学ぶユニークな形をした博物館。歴史が好きで東京が好きなら欠かせないスポットです。当時の江戸文化を再現したジオラマだけではなく、明治から昭和、現代の東京の変遷が学べます。常設展だけではなく企画展も別料金で楽しめます。近くには両国国技館も。
■江戸東京博物館
・アクセス：首都高速6号 向島線駒形出口、7号小松川線錦糸町出口より約10分
・住所：東京都墨田区横網1-4-1
・駐車場：2時間まで1000円
・URL： https://www.edo-tokyo-museum.or.jp/
地図を見る

江戸東京たてもの園（小金井市）

via www.gettyimages.com
「江戸東京博物館」が楽しめたなら、江戸東京たてもの園もぜひ寄りたいスポットです。小金井市の小金井公園内にあるスポットで、むかしの日本の文化財となる建物が移築されています。ジブリ映画『千と千尋の神隠し』の元となった銭湯や商家などが残されているほか、2.26事件の現場にもなった高橋是清邸などの建築物や明治から昭和時代の都電、万世橋交番など見どころがたくさんあります。
■江戸東京たてもの園
・アクセス：五日市街道沿い、「小金井公園前」信号より公園へ
・住所：東京都小金井市桜町3-7-1（都立小金井公園内）
・駐車場：あり（有料）
・URL： http://tatemonoen.jp/
地図を見る

殿ヶ谷戸庭園（国分寺市）

国分寺駅から徒歩で2分という近さにある日本庭園。大正初期に江口定條の別荘として創建されたものが昭和になって三菱財閥の岩崎家の所有となり、その後東京都の所有と変わりました。武蔵野の面影が残る自然と湧水、紅葉の美しさが見どころです。
■殿ヶ谷戸庭園
・アクセス：JR中央線、西武国分寺線・西武多摩湖線「国分寺」下車　徒歩2分
・住所：東京都国分寺市南町二丁目
・駐車場：なし
・URL： http://teien.tokyo-park.or.jp/contents/index036.html
地図を見る

深大寺（調布市）

via www.gettyimages.com
武蔵野の自然が残る深大寺で散策をゆったり楽しんではいかがでしょう。境内でお詣りをしたり、大師堂や釈迦堂を見て回ったり。お蕎麦屋さんがたくさんあるので昼食にそばを食べてみたり。鬼太郎の作者水木しげるのゆかりの地でもあり、「鬼太郎茶屋」は外せないスポットです。となりの神代植物公園も広大で見ごたえあります。
■深大寺
・アクセス：中央自動車道調布ICから車で2.3km
・住所：東京都調布市深大寺元町5-15-1
・駐車場：なし（近隣の有料駐車場利用）
・URL： http://www.jindaiji.or.jp/
